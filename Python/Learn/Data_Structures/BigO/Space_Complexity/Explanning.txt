Space Complexity คือ Algorithm ของเราต้องการ หน่วยความจำ(Ram)
เพิ่มขึ้นหรือมากน้อยเพียงใด เมื่อขนาดของข้อมูลนำเข้าเพิ่มมากขึ้น

ประเภทของ Space Complexity
1.Fixed Part  = พื้นที่ที่ใช้เก็บตัวแปรพื้นฐาน,คงที่ หรือ ส่วนประกอบของโปรแกรม
2.Variable Part = พื้นที่ที่ใช้เก็บโครงสร้างของข้อมูล เช่น List Dictionary ที่ขยายขนาดตาม n
